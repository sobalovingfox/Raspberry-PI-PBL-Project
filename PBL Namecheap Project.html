Raspberry Pi Project

So for a school project, I decided to make a Raspberry Pi computer/laptop. This was new frontiers for me as I have
literally never touched Raspberry Pi in my life before, so this was a pretty fun project to work on. This project taught
me a lot about emulation and mini computers along with the coding that came when building this website. If you scroll
down on this website, there will be a video describing the process and completion of this project in more depth.

Software and Controllers

First of all, the Raspbian OS is required to start the screen which requires a SD card. Once the screen and software like touchscreen has been
downloaded on the Linux Terminal, the second step will be to get the actual emulator. I had many choices for this emulator such as RecalBox,
Batocera, Lakka, RetroPie, and RetroArch. I decided to use Recalbox because of the ease of use and installing. The third step would to burn the
emulator image through RPI Imager which you would have to used to install the Raspbian OS originally, for more information about this imager
check RPI Imager For controllers, I used my Switch Pro Controllerbut most controllers should work, check this website for details on Controller
Support. Once the software has been downloaded, you need to configure your controller. Then once you download your games onto your SD card
or flash drive, you can play! If you are using any of the Raspberry Pi series as your source of power for your emulator, don't expect to play PS3 or
Xbox 360 games cause those games will most likely damage the Pi. Depending on which emulator you use, you can customize the overlays or add
media through the Kodi Media application which lets you add songs that will play on the home screen. That should be all the software you will need
to make your best RPI emulation station to life!
